# E-Commerce-Website-Using-PHP-MYSQL

## Summary
This is my Introduction to database project. Which is complete Ecommerce Website. Anybody can use this project for learning purpose

## Platform Used
### Front-End
  (i) HTML5 <br>
  (ii) CSS3 <br>
  (iii) JavaScript <br>

### Back-End
  (i) PHP <br>
  (ii) MySQL <br>

## Key Features
### Public User
(i) Search Product <br>
(ii) View Product <br>
(iii) Create User Account <br>

### Signin User
(i) Search Product <br>
(ii) View Product <br>
(iii) Create Order <br>
(iv) Change Email & Password <br>
(v) Can View Previous Order with UPDATE and DELETE <br>

### Admin
(i) Add New Product <br>
(ii) Update Product <br>
(iii) Delete Product <br>
(iv) Confirm Order <br>


### Visitors
(i) Complete Shopping 
(ii) Manage there shopping cart

## Conclusion
There are also many more feature which are not in the list. Feel free to use this project
